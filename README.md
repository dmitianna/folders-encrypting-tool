# Folder Encrypting Tool

Консольная утилита на C++ для шифрования и дешифрования файлов в указанной папке и её подпапках.

Программа выполняет рекурсивный обход директории и защищает файлы с использованием криптографического алгоритма AES-256.

---

## Функционал

Программа поддерживает два режима работы.
