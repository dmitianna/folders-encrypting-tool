# Folder Encrypting Tool

---
## Постановка задачи
Требуется реализовать консольную утилиту на языке C++  для шифрования и дешифрования файлов в указанной папке, её подпапках. Также возможно шифрование одиночного файла при указании пользователем пути к файлу. Программа выполняет рекурсивный обход введенной пользователем директории,  по запросу пользователя выполняет либо шифрование с использованием AES-256 GCM, либо расшифрование.

## Функционал
Программа поддерживает два режима работы.
### Режим шифирования
В программе реализовано шифрование файлов с использованием комбинации алгоритмов PBKDF2 (HMAC-SHA256) и AES-256 в режиме GCM.<br>
При шифровании выполняются следующие действия:<br>
- генерируется случайная соль (16 байт), используемая для генерации ключа<br>
- на основе пароля пользователя и соли с помощью PBKDF2 (100000 итераций) формируется криптографический ключ<br>
- создаётся случайный вектор инициализации (IV) длиной 12 байт<br>
- исходные данные шифруются с использованием AES-256-GCM<br>
- одновременно формируется аутентификационный тег (16 байт), обеспечивающий контроль целостности<br>

Результат записывается в файл с использованием QSaveFile, что предотвращает повреждение данных при сбоях.
### Режим дешифрования
При дешифровании выполняются обратные действия:<br>
- проверяется наличие сигнатуры зашифрованного файла<br>
- из файла извлекаются соль, IV и зашифрованные данные<br>
- по введённому паролю и сохранённой соли повторно вычисляется ключ<br>
- выполняется расшифрование с использованием AES-256-GCM<br>
- на завершающем этапе проверяется аутентификационный тег<br>
Если пароль неверный или файл был изменён, проверка тега завершается ошибкой, и операция прерывается без изменения файла<br>
## Инструкции для пользователя
После запуска программа работает в интерактивном режиме.

Доступные команды:<br>
- encrypt — шифрование файла или папки<br>
- decrypt — дешифрование файла или папки<br>
- exit — выход из программы<br>

**Пример использования**<br>
Enter command (encrypt/decrypt/exit): encrypt<br>
Enter file or folder path: C:\test\file.txt<br>
Enter password: 1234<br>

## Ограничения
Скрытые файлы и папки не обрабатываются<br>
Символьные ссылки игнорируются<br>
Системные файлы и каталоги не обрабатываются<br>
Максимальная длина пароля — 64 символа<br>
При потере пароля восстановление данных невозможно<br>
